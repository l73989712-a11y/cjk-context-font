# Changelog

## 0.4.0

- 增加全局和网站级个人词典。
- 增加完全/包含匹配和右键快捷操作。
- 增加识别试验台与可解释数据属性。
- 增加黄金语料、发布安全检查和开源仓库模板。
- 保持 0.2.2 起的稳定 DOM 默认值。

## 0.3.0

- 抽出无 DOM 依赖的 `core.js` 识别模块。
- 为识别理由与评分增加单元测试。
- 增加当前标签页运行诊断和可复制报告。
- 增加动态 DOM、稳定模式和 ruby 兼容的浏览器回归测试。
- 增加静态检查、测试和打包脚本。
- 保持 0.2.2 稳定 DOM 模式为默认行为。
- 修正升级迁移：只有从 0.2.2 之前版本升级时才强制安全设置。

## 0.2.2

- 默认稳定模式，不再自动改写文本节点结构。
- 跳过 ruby 与片假名注音脚本生成节点。
- 合并动态 Mutation，修复页面卡顿与脚本冲突。
