# 识别模型

## 原则

扩展优先使用明确标记，自动模型只处理缺少标记的文本。纯汉字中日识别不可能保证绝对正确，因此模型同时输出置信度、证据、得分和决策差值。

## 优先级

1. 临时手动标记；
2. 元素站点规则；
3. 个人词典；
4. 距离很近的 `lang`；
5. 假名、韩文等文字系统证据；
6. 简繁特征、日文特有汉字、常见短语和标题语境评分；
7. 页面语言、域名和用户回退偏好。

## 三档策略

- **保守**：提高纯汉字得分和领先差值门槛，适合最重视低误判率的用户。
- **均衡**：默认值，在识别能力和误伤之间折中。
- **积极**：降低门槛，更愿意识别缺少假名的 ACG 标题。

三档策略不会削弱手动标记、站点规则、词典、局部 `lang`、假名或韩文等明确证据。

## 标题语境

`新世界`、`青空`、`花火` 等词在中日文中都可能出现。0.6.0 将这些词视为弱标题证据：只有元素本身是标题、站点适配器把它标为标题，或试验台明确选择“标题”时，才显著提高日文得分。

`紅蓮華`、日文特有字形和用户词典属于更强证据，不完全依赖标题语境。

## 中日混合整句

稳定模式不会拆分连续文本节点。如果一条明显的中文评论中只有少量片假名，例如“这个视频里的ゲーム翻译得很好”，把整句设为日文字体反而会损害更多中文汉字。因此模型会在中文短语证据充足且假名密度较低时，选择中文作为整句主语言。

## 误判报告

设置页试验台可复制 JSON 报告，其中包含用户主动输入的测试文字、模拟语境、文字系统统计和识别结果，不会读取其他网页内容。
